Controls

This game is for two players. If you download it and only have one controller, you will be disappointed.

Move - Left Stick

Jump - A or LT

Aim - Right Stick

Shoot - RT

Pause - Start

Skip Dialogue/Cutscene - B

What is This?

This is an original game made in a team of 4 in roughly 1 month. The objective was to build and release a complete, novel, interesting game, including marketing materials.

How Was it Made?

This game is built on Unity and C#. I especially worked on level design, enemy design, UI, some game design concepts, and miscellaneous other subsystems, such as player health. Additionally, I made art for the level tileset and all of the enemies. I also wrote the Pulitzer-prize worthy dialogue, and edited the Oscar-worthy game trailer, found on the itch.io page. Aside from that, I also used Jira for project management, git for version control, and Aseprite for sprite creation/modification.
